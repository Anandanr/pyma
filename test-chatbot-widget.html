📝 How to Embed on Your Website

Clients just need to add this one line of code to their website:

<script src="https://widget.pym.ink/chatbot.js" data-api-key="pyma_afbc44a5899b4cbeb1cbed4dea4c20ad"> </script>

That's it! The widget will:

  • Appear in the bottom-right corner
  • Start handling customer conversations instantly
  • Learn from your knowledge base
  • Escalate complex issues to humans if needed
  • Work on all modern browsers
